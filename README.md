# homebrew-katazuke

Homebrew tap for [katazuke](https://github.com/agrahamlincoln/katazuke), a developer workspace maintenance tool for tidying up git repositories.

## Installation

```bash
brew tap agrahamlincoln/katazuke
brew install katazuke
```

## Updating

```bash
brew update
brew upgrade katazuke
```

## Source

This tap installs pre-built binaries from the [katazuke releases](https://github.com/agrahamlincoln/katazuke/releases). For source code, documentation, and issue tracking, see the [main repository](https://github.com/agrahamlincoln/katazuke).
